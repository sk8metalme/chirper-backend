# Product Overview

chirper-backendは、Twitter風SNSアプリケーション「Chirper」のバックエンドサービスです。Spring BootベースのREST APIサーバーとして、ユーザー管理、ツイート管理、タイムライン、ソーシャル機能のビジネスロジックを提供します。

## Core Capabilities

1. **認証・ユーザー管理**: JWT認証によるステートレス認証基盤、ユーザー登録・ログイン・プロフィール管理
2. **コンテンツ管理**: ツイート投稿・取得・削除、280文字制限、論理削除による安全なデータ管理
3. **タイムライン機能**: フォローユーザーのツイートを時系列で表示、N+1クエリ対策による高速レスポンス
4. **ソーシャル機能**: ユーザーフォロー、ツイートいいね・リツイート、ユーザー・ツイート検索
5. **セキュアな基盤**: bcryptパスワードハッシュ化、JWT認証、SQLインジェクション対策、セキュリティヘッダー設定

## Target Use Cases

- **フロントエンドサービスとの連携**: chirper-frontendが主要なクライアント、WebブラウザからのアクセスをREST APIで処理
- **リアルタイムソーシャル体験**: タイムライン表示、フォロー管理、いいね・リツイートによるユーザーエンゲージメント
- **スケーラブルなSNS基盤**: ステートレス設計、水平スケーリング対応、95パーセンタイル500ms以下の高速レスポンス

## Value Proposition

- **オニオンアーキテクチャ**: ドメイン駆動設計に基づく明確な責務分離、保守性とテスタビリティの高さ
- **パフォーマンス重視**: N+1クエリ対策、インデックス最適化、接続プール管理による高速レスポンス
- **セキュリティファースト**: JWT認証、パスワードハッシュ化、入力検証、セキュリティヘッダー設定
- **高品質コードベース**: 単体テストカバレッジ95%以上、明確なエラーハンドリング戦略、構造化ログ
